CSS Selector, Combinators, Combine Reference

Table Selector

Tag Name Description
#nameid id Selects the element with a specific id. You can also combine the ID selector with the type selector.
.nameclass { } class Select elements by their class
* { } Universal Select everything!
A { } Element name Select the element selector by name that element directly
a[target=”value”] { } Attribute Value Selector Select all elements that have a specific attribute value

Table Combinators

Tag Name Description
A, B Comma Combinator Combine, selectors, with... commas!
A + B Adjacent Sibling Selector Select an element that directly follows another element
A ~ B General Sibling Selector Select elements that follows another element
A > B Child Selector Select direct children of an element
A B Descendant Selector Select an element inside another element

Table Combine Selectors

Tag Name Description
A.className Combine the Class Selector You can combine the class selector with other selectors, like the type selector.
A * Combine the Universal Selector This selects all elements inside of A.
#id A Combine the Descendant & ID Selectors You can combine any selector with the descendent selector.
